# Olive Clone

Responsive React + Vite demo built with Tailwind CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Git Notes

- `node_modules/` is ignored and should not be pushed.
- `dist/` is ignored so build output stays out of the repo.
- Environment files like `.env` are ignored too.

## Project Structure

- `src/App.jsx` - main UI
- `src/main.jsx` - app entry point
- `src/index.css` - global styles
- `index.html` - Vite entry HTML
